
## Framework

- Built using PHP Laravel: https://laravel.com/docs/8.x/installation


## Installation

- Install Laravel CLI
- Download the repo
- On terminal `cd /project_folder` and run the following command `php artisan serve`
- View the API by going to http://127.0.0.1:8000/api/employees/
